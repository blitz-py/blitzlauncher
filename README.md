## Overview

blitz.gui is an open-source game launcher with an integrated emulator. It consolidates and organizes games from various platforms, providing users with a centralized and customizable interface for managing their game library. The launcher allows you to not only organize your games but also play them seamlessly using the built-in emulator.

## Features

- **Platform Integration:** Automatically detects and lists games from popular platforms like Steam, Epic, etc.
- **Local Game Discovery:** Scans specified folders for locally installed games.
- **User-Friendly Interface:** Intuitive design with customizable styles, fonts, and other preferences.
- **Game Information:** Displays relevant details, such as game names, original installation sizes, and more.
- **Launch Games:** Easily launch games directly from the launcher.
- **Integrated Emulator:** Play supported games seamlessly within the launcher using the built-in emulator.
- **Resource-Efficient:** Designed to utilize low RAM, ensuring smooth performance even on systems with limited resources.
